HTTP Header Manipulation

This test looks to see if the HTTP headers sent by a probe are being altered when they are transmitted to a control back-end server run by OONI. This is the sign of a transparent HTTP proxy manipulating the traffic of a probe.

(HTTP headers: When you access a website, the HTTP protocol defines what you need to send so that you can connect to a website. You need to send different “fields”, such as the path that you want to access, the method of the request, etc.)
