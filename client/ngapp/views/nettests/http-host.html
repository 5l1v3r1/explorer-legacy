What does this test do?

This test attempts to (1) detect the presence of a middle-box (something that is in between the network path of the probe and a control server run by OONI) and (2) assess which censorship circumvention techniques may be able to bypass the censorship implemented by the middle-box. This can lead sometimes to identifying the type of infrastructure being used to implement censorship.
