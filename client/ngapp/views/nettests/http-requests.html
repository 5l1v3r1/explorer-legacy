What does this test do?

This test requests a website first over the network of the user and then over the Tor network. The two results are compared and if the length of the body of the websites differs by some percentage, then we consider that there is an anomaly. Other signs of anomalies can include the fact that the request over the user's network fails or that the headers do not match.

Note: False positives might occur when the Tor control connection is being discriminated by the server. This happens, for example, when a CloudFlare CAPTCHA page appears.
