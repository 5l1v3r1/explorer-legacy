Open Observatory of Network Interference (OONI), is a global community of network probes that measure the internet to identify "network anomalies" that can be signs of internet censorship or surveillance.

Using the map on this page you can see where measurements are run and in which countries we have identified instances of internet censorship and particular types of censorship software or vendors.
